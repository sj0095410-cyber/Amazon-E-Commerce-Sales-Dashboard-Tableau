# Amazon E-Commerce Sales Dashboard (Tableau)

## Project Overview

This project presents an interactive Amazon Sales Dashboard built using Tableau. The dashboard provides insights into sales performance, order trends, product categories, fulfilment methods, and regional sales distribution.

The goal of this project is to analyze sales data and create a business-friendly dashboard that helps stakeholders monitor key performance indicators and identify trends.

---

## Key Performance Indicators (KPIs)

* Total Sales
* Total Orders
* Total Quantity Sold
* Average Order Value (AOV)

---

## Dashboard Insights

### Monthly Sales Trends

Tracks sales performance over time and identifies growth patterns.

### Order Status Analysis

Shows the distribution of orders across different statuses such as shipped, cancelled, returned, and pending.

### Sales by State

Highlights the top-performing states based on total sales.

### Sales by Category

Compares sales performance across product categories.

### Fulfilment Analysis

Analyzes order fulfilment methods and their contribution to total orders.
